# 🎯 Hangman Pro

A modern Hangman game built with HTML, CSS, and JavaScript.

## 🌟 Features

- Modern glassmorphism UI
- Responsive design
- Multiple word categories
- High score system
- Win/Lose modal screens
- Keyboard interaction
- GitHub Pages compatible

## 🎮 How to Play

1. Open the game.
2. Guess letters by clicking the keyboard.
3. Complete the word before running out of lives.
4. Earn points for every word solved.
5. Try to beat your high score.
